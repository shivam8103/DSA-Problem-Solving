## **Problem Statement:** - Given 2 sorted array. Return resultant array with intersection of these 2 array. (Duplicate allowed)

### **Example: 1**

**Input:**

``arr1 = { 1, 1, 2, 3, 3, 4, 4, 5, 6 };``

``arr2 = { 1, 3, 3, 4, 6 };``

**Output:**  ``{1,3,3,4,6}``

**Explanation:** Duplicate is allowed, so we can add 3 twice.
